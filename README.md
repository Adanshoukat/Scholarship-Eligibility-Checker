# Scholarship-Eligibility-Checker
# CampusScholar - Advanced Eligibility Portal

A professional desktop graphical user interface (GUI) software built with Python and PyQt5. This application serves as an automated, rule-based expert system designed to evaluate and log student eligibility for various university scholarships dynamically.

# Core Features

- **Modern UI/UX Dashboard:** Designed using a custom premium warm theme (Warm Ivory, Terracotta Orange, and Deep Indigo accents) with robust field alignments and clear visibility.
- **Rule-Based Scoring Engine:** Computes eligibility instantly based on multiple strict institutional criteria (CGPA boundaries, Family Income caps, and a minimum 75% Attendance threshold).
- **Live Analytics Panel:** Real-time counters built directly into the interface to monitor the total number of applications processed and the total successful candidates during the session.
- **Automated Data Reporting:** A custom reporting engine that automatically logs student data (Registration Number, GPA, Income, Attendance) and the system's final decision into a structured `scholarship_report.csv` file inside the local directory.

## Installation & System Setup

### Prerequisites
Make sure you have Python installed on your system. You will need to install the PyQt5 framework before running the software.

### Steps to Run

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/Adanshoukat/Scholarship-Eligibility-Checker.git](https://github.com/Adanshoukat/Scholarship-Eligibility-Checker.git)
   cd Scholarship-Eligibility-Checker
   ```

## Install Dependencies:
Open your integrated terminal and install the required graphical components:

```bash
pip install PyQt5
```

## Launch the Software:
Run the primary execution script:

```bash
python scholarship_checker.py
```
